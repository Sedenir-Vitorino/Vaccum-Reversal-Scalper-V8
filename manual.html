🎯 What is Vacuum BTCUSD

A reversal scalper that identifies candle patterns and filters entries by trend direction and price position relative to support and resistance zones. Developed exclusively for BTCUSD CFD on the M1 timeframe.

Key Features

  • Candle Patterns: Detects 7 classic patterns Active by default
  • Trend Filter (EMA): Defines market direction Disabled by default
  • Support & Resistance: Dynamic S/R from last N candles Disabled by default
  • ATR Filter: Blocks entries when volatility is outside normal range Active by default
  • Dynamic SL: SL rises automatically Disabled by default
  • SL/TP at Broker: Always registered as absolute prices Always active

🔧 Installation

  1. Open cTrader → Automate → New cBot. Name it VacuumBTCUSD.
  2. Paste the code from VacuumBTCUSD.cs. Press F6 to compile.
  3. Open a BTCUSD M1 chart. Drag the cBot to the chart.
  4. Import optimized parameters (JSON file).
  5. Always test on Demo account first!

✅ SL/TP Confirmation

After opening each order, the log should show: SL/TP aplicados (tentativa 1) ✓ | SL=84360.00 TP=84550.00. This confirms protection is active.

⚠️ BTCUSD Only

This bot is calibrated exclusively for BTCUSD CFD. Using on other assets will produce incorrect SL/TP values.

💡 Recommendation

Start with default parameters on a demo account for 1-2 days. Monitor the log with ShowDebug: true before going live.
